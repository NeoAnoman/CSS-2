Advanced Selectors


Reference: https://www.w3schools.com/cssref/css_selectors.asp


  • * => Selects all elements (not used quite often, overrides everything except ids)

  • Tag Selector => Selects specific tag

  • .Class Selector => Selects specific class

  • #ID Selector => Selects specific id

  • Element,Element => Selects more than 1 element

    • Hello

    World

  • Element1 Element2 => Selects element1 containing element 2

    • I am child of b tag


  • :Hover => Adds style when we hover over the element

  • ::Selection => Adds style when text is selected

  • Element1+Element2 => Selects all element2 elements that are placed immediately after element1 elements

    • Hello


  • [attribute1] => Adds style to those elements containing attribute1
